O quê é HTML?

HTML é uma linguagem de marcação de texto, e Não é uma linguagem de programação.

Ela é utilizada para realizar a formatação dos textos e como será a disposição básica deles. De uma forma figurada, utilizando o corpo humano como exemplo, o HTML é o esqueleto, é a parte que da a forma ao site, porém ela fica abaixo de outra camada, uma camada estética, que no caso será o CSS.
